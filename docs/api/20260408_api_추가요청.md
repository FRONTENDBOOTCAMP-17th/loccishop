# API 추가 요청 명세서

## 1. Ritual Steps

### GET /api/products/:productId/ritual-steps

### 설명

특정 상품이 속한 리추얼 세트 정보를 반환합니다.
해당 상품에 리추얼 세트가 없는 경우 `steps`는 빈 배열로 반환됩니다.

---

### Request

| 항목   | 내용                                    |
| ------ | --------------------------------------- |
| Method | GET                                     |
| URL    | `/api/products/:productId/ritual-steps` |

**Path Parameter**

| 파라미터  | 타입   | 필수 | 설명    |
| --------- | ------ | ---- | ------- |
| productId | number | ✅   | 상품 ID |

---

### Response

**성공 (200 OK)**

| 필드                 | 타입           | 설명                                   |
| -------------------- | -------------- | -------------------------------------- |
| title                | string         | 리추얼 세트 제목                       |
| description          | string \| null | 리추얼 세트 설명 (없으면 null)         |
| steps                | array          | 리추얼 단계 목록 (없으면 빈 배열 `[]`) |
| steps[].step         | number         | 단계 순서 (1, 2, 3...)                 |
| steps[].product_id   | number         | 단계 상품 ID                           |
| steps[].product_name | string         | 단계 상품명                            |
| steps[].volume       | string \| null | 용량 (없으면 null)                     |
| steps[].price        | number         | 단계 상품 가격 (원)                    |
| steps[].image_url    | string         | 단계 상품 이미지 URL                   |

---

### Response 예시

**steps가 있는 경우**

```json
{
  "title": "부드러운 손, 윤기 나는 손톱",
  "description": "이제 3단계 핸드 & 네일 리추얼을 경험해 보세요.",
  "steps": [
    {
      "step": 1,
      "product_id": 101,
      "product_name": "시어 핸드 스크럽 (카리테 콩포르)",
      "volume": "30 ml",
      "price": 19000,
      "image_url": "https://example.com/images/product-101.jpg"
    },
    {
      "step": 2,
      "product_id": 102,
      "product_name": "시어 버터 핸드 크림 (카리테 콩포르)",
      "volume": "75 ml",
      "price": 29000,
      "image_url": "https://example.com/images/product-102.jpg"
    },
    {
      "step": 3,
      "product_id": 103,
      "product_name": "시어 네일 앤 큐티클 오일 (카리테 콩포르)",
      "volume": "7.5 ml",
      "price": 26000,
      "image_url": "https://example.com/images/product-103.jpg"
    }

}
```

**steps가 없는 경우**

```json
{
  "title": "라벤더 클린 핸드 케어",
  "description": null,
  "steps": []
}
```

---

### 프론트엔드 처리 방침

- `steps`가 빈 배열(`[]`)인 경우 리추얼 섹션을 **렌더링하지 않음**
- `description`이 `null`인 경우 설명 텍스트 영역을 **표시하지 않음**
- `steps` 개수는 2개 또는 3개로 가변적으로 처리

## 2. 상품 등록 API 필드 추가 요청

### POST /api/loccishop/v1/admin/products

기존 상품 등록 API의 Request Body에 아래 두 필드 추가를 요청합니다.

---

### 추가 필드

| 필드          | 타입   | 필수 | 설명                                                      |
| ------------- | ------ | :--: | --------------------------------------------------------- |
| `productInfo` | object |  X   | 상품 상세 정보 {ingredients, howToUse, productDisclosure} |
| `options`     | array  |  X   | 용량 옵션 목록 (같은 라인의 다른 용량 상품 연결)          |

**productInfo 상세**

| 필드                            | 타입   | 필수 | 설명              |
| ------------------------------- | ------ | :--: | ----------------- |
| `productInfo.ingredients`       | string |  X   | 성분              |
| `productInfo.howToUse`          | string |  X   | 사용법            |
| `productInfo.productDisclosure` | string |  X   | 상품정보 제공고시 |

**options 상세**

| 필드                  | 타입   | 필수 | 설명                    |
| --------------------- | ------ | :--: | ----------------------- |
| `options[].size`      | string |  O   | 용량 표기 (예: "75 ml") |
| `options[].productId` | number |  O   | 해당 용량 상품의 ID     |

> options가 있는 경우, 각 옵션 클릭 시 해당 `productId`의 상품 페이지로 이동합니다.
> price, stock은 각 상품에서 개별 관리하므로 options에 포함하지 않습니다.

---

### 추가 후 Request Body 예시

```json
{
  "name": "시어 버터 핸드 크림 (카리테 콩포르)",
  "shortDescription": "가벼운 수분 핸드 크림",
  "description": "상품 상세 설명입니다.",
  "price": 29000,
  "discountPrice": 25000,
  "stock": 100,
  "categoryId": 5,
  "badge": "new",
  "images": {
    "representative": "https://example.com/uploads/rep.jpg",
    "mainSlides": ["https://example.com/uploads/slide1.jpg"],
    "detailContents": ["https://example.com/uploads/detail1.jpg"]
  },
  "productInfo": {
    "ingredients": "정제수, 시어버터, 글리세린 ...",
    "howToUse": "적당량을 덜어 손에 부드럽게 펴 바릅니다.",
    "productDisclosure": "1. 내용물의 용량 및 중량: 75ml"
  },
  "options": [
    { "size": "75 ml", "productId": 102 },
    { "size": "150 ml", "productId": 205 }
  ]
}
```
